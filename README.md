bcrypt | 비밀번호 암호화

dotenv | 포트번호, DB비밀번호 등 암호화

jsonwebtoken | 회원가입 작동 방식
moment | 날짜 포맷 설정

mongoose | 몽고DB
morgan | console 원활한 확인
multer | 이미지 데이터
nodemon | 서버 재실행
xml-js | XML > JSON 형식변환
